## git add

![](https://cdn-icons-png.flaticon.com/512/114/114888.png)


**git add *[файл]*** - добавляет файл в индекс. 

Чтобы добавить все файлы в каталоге в индекс (кроме игнорируемых), используйте команду:

```bash=
git add . 
```
